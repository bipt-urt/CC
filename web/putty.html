示例一个简单的Hello World程序：
1、新建一个程序命名为Hello.c
vim Hello.c
2、进入vim的编辑模式（按下i后即可进行代码编辑）
i
3、编辑代码：
#include int main()
{
printf("Hello World\n");
return 0;
}
4、进入vim命令模式（按一下Esc）
Esc
5、对代码进行保存并退出编辑器：
:x
6、编译代码（注意不能加后缀名.c）
make Hello
7、执行代码（注意还是不能有后缀名.c）
./Hello
